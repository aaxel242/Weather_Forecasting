# Explicación Carpeta Utils

Aquí van las funciones auxiliares y reutilizables que sirven de apoyo a los scripts o módulos principales.

- funciones para leer/escribir archivos. (file_utils.py)
- funciones para generar gráficos. (plot_utils.py)
- funciones matemáticas recurrentes. (math_utils.py)
- funciones para conectarse a una base de datos. (db_utils.py)

- Características:
- No se ejecutan solos, se importan desde scripts u otros módulos.
- Contienen lógica genérica y modular.

Ayuda:

- Piensa en ellos como caja de herramientas que los scripts usan.
